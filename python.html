Automate with Python

by Nishant Sahoo and Harshini N

Duration: approximately 2 weeks

Overview

Python is a popular programming language created in 1991 by Guido van Rossum. It is used for server-side web development, software development, mathematics, system scripting and many more.
It is one of the most preferred languages mainly because of its simple syntax, syntax that allows developers to write programs with fewer lines than most programming languages and the vast amount of libraries available making writing and reading programs easier.
Knowledge of Python helps a lot for those who are aiming at a career in the field of Computer Science. And even for the others, learning Python is fun and can be used to code programs that will help automate daily, mundane work.
In this mini mentorship drive, we aim to help students learn how to use Python and its libraries efficiently.

Goals

  • Learn the programming language, Python
  • Learn how to use Python to write programs that do in minutes what would take one hours to do by hand
  • Create Python programs that effortlessly perform useful and impressive feats of automation
  • Learn how to use the available Python libraries

Milestones

  • Lists, strings, dicts, tuples, Pattern matching
  • Reading and writing files and Organising files (Create, update, move, and rename files and folders)
  • Working with CSV files and JSON data
  • Web scraping, Search the Web and download online content
  • Update and format data in Excel spreadsheets of any size (Optional)
  • Split, merge, watermark, and encrypt PDFs (Optional)
  • Send reminder emails and text notifications
  • Scheduling tasks and launching programs
  • Twitter login, tweets
  • Manipulating images (Optional)
  • Controlling the keyboard and mouse with GUI automation (Optional)
