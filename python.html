Automate with Python

by Nishant Sahoo and Harshini N

Duration: approximately 2 weeks

Overview

Python is a popular programming language created in 1991 by Guido van Rossum. Python is used for all sorts of things like building websites, web scrapers, crawlers, APIs, automated and messaging bots, data analysis, machine learning, and natural language processing. The language is designed to be easy to read, while still being very powerful, which makes it a great language for beginners to learn.
Why should one learn Python? It is easier to learn and thereâ€™s a lot of demand for Python Programmers. And one can do some really cool things with Python from data science to building websites.
In this mini mentorship drive which would last for about 12-14 days, we aim to help students learn how to use Python and its libraries efficiently to help automate daily mundane work.

Goals

  • Learn the programming language, Python
  • Create Python programs that effortlessly perform useful and impressive feats of automation.
  • Learn how to use the available extensive Python libraries

Milestones

  • Basics of Python (Functions, Lists, Dictionaries, Strings, Pattern Matching
  • Basics of Python (Reading and writing files)
  • Basics of Python (Organising files)
  • Split, merge, watermark, and encrypt PDFs and Word docs
  • Web scraping. Ex: Automate Facebook/Twitter/SLCM login, tweeting image/text.
  • Send reminder emails and text notifications, Scheduling tasks and launching programs
  • Manipulating images
  • Controlling the keyboard and mouse with GUI automation
